## 📦 安装
通过npm

```bash
npm install ninecat-ui --S
```
通过yarn

```bash
yarn add ninecat-ui
```

## 🔨 快速开始

```javascript
import Vue from 'vue'
import Ninecat from 'ninecat-ui'
import 'ninecat-ui/package/main.css'

Vue.use(Ninecat)

// or
import {
  Button
  // ...
} from 'ninecat-ui'

Vue.component(Button.name, Button)

```

## ⌨️ 开发

本地Clone

```bash
$ git clone https://github.com/ninecat-ui/ninecat-ui.git
$ cd ninecat-ui
$ yarn install
$ yarn start
```
