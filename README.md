# ninecat-ui

A Vue.js 2.0 UI Toolkit  that can be previewed on the mobile side.

## install devDependencies

`npm install`

## run dev

`npm run dev`

## run prod

`npm run prod`

## run test 

`npm run test`

## Browser Support
Modern browsers and Internet Explorer 10+.
