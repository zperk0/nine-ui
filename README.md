## üì¶ Install

Use npm

```bash
npm install ninecat-ui --S
```

Use yarn

```bash
yarn add ninecat-ui
```

## üî® Quick Start

```javascript
import Vue from 'vue'
import Ninecat from 'ninecat-ui'
import 'ninecat-ui/package/main.css'

Vue.use(Ninecat)

// or
import {
  Button
  // ...
} from 'ninecat-ui'

Vue.component(Button.name, Button)

```
