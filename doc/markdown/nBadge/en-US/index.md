
# Badge

Used for remind.

## When To Use

Use it where you need news to remind.

### Examples

:::demo
```html
<template>
  <n-badge>99+</n-badge>
  <n-badge type="new">
    New
  </n-badge>
  <n-badge type="small" />
</template>
```
:::

### API

| Property | Description | Type | Default |
| :--- | :--- | :--- | :--- |
| type | there are three types:`default`,`new`,`small` | String | default |
