
# Input

Used for input.

## When To Use

Use it where you need input.

### Examples

:::demo
```html
<template>
  <n-input size="md" />
</template>
```
:::

### API

| Property | Description | Type | Default |
| :--- | :--- | :--- | :--- |
| size | Input box size | String | middle |
| disabled | Whether the input is disabled. | Boolean | false |
