# Steps 步骤条
引导用户按照流程完成任务的导航条。

- Steps 步骤条组件
- StepsItem 步骤条内的每一个步骤。

## 代码演示

### 基本使用

:::demo
```html
<template>
  <n-steps>
    <n-step title="步骤1" description="步骤1描述"/>
    <n-step title="步骤2" description="步骤2描述"/>
    <n-step title="步骤3" description="步骤3描述"/>
    <n-step title="步骤4" description="步骤4描述"/>
  </n-steps>
</template>
```
:::
